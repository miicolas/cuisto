# VR Sandbox

Projet VR Unity où le joueur évolue dans une salle interactive et peut manipuler des objets à la volée.

## La salle

- Sol, murs et plafond délimitant l'espace jouable
- Éclairage avec un **interrupteur** pour allumer et éteindre les lumières
- **Porte d'entrée** qui s'ouvre automatiquement en détectant la position du joueur

## Contrôles et interactions

- **Spawn d'objets** : input dédié pour faire apparaître des objets (table, chaise, ...)
- **Reset de la scène** : input qui remet la scène dans son état initial
- **Menu de sélection** : input qui ouvre une interface permettant de choisir l'objet à faire apparaître
- **Manipulation d'objet** : en ciblant un objet et en maintenant la touche enfoncée, le joueur peut
  - déplacer l'objet (modification de la position)
  - le faire pivoter (rotation)
  - le supprimer
